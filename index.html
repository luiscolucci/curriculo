Rastreador de Software para desinstalação

Programa desenvolvido em Python

O fluxo de trabalho é este: O script pede seu login, senha e nome do software. Ele descobre automaticamente as redes e varre todos os hosts ativos. Ele começa a verificar cada host individualmente. No momento em que encontra o software instalado em um host específico, o processo no terminal para temporariamente e exibe a pergunta: Deseja desinstalar o software de [endereço_de_ip_do_host]? [s/N]: Nesse ponto, você decide: Digite S e pressione Enter para confirmar a desinstalação naquele host. Pressionar Enter (ou digitar qualquer outra coisa) para não desinstalar ele pular para o próximo host. Ele faz isso para cada máquina onde o software é encontrado, dando a você o controle total, host por host, no momento da descoberta.
